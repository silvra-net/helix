Helix block explorer

Helix (HLX) is a post-quantum Layer-1 blockchain in Rust — every signature is NIST FIPS 204 (ML-DSA-65), consensus is proof-of-stake BFT with ~2s blocks. Everything below is read live from this node, no third party in between. New here? Start with the README.

Signatures ML-DSA-65 Hash BLAKE3 PoS + BFT ~2s blocks Cap 33M HLX Node …
Public test network. HLX here is a test token with no monetary value. The chain can be reset from genesis and the validator set is small, so it tolerates zero faults today — decentralization is the goal, not yet the state. No external security audit. Explore it, run a node, build against it — just don't hold anything you'd mind losing.
